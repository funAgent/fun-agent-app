## 我们是谁

**funAgent** 是一个致力于让 AI Agent 变得有趣的开源组织。

我们相信，与 AI 协作编程不应该只是效率工具，更应该是一种令人愉悦的体验。通过开源工具、深入浅出的教学课程和创意实验，我们希望让每个开发者都能感受到 AI 编程的乐趣。

---

## 🎯 愿景

> "让 AI Agent 变得有趣"

- **工具** — 构建实用的 AI 编程工具，让开发者的日常工作更有趣
- **课程** — 创建体系化的学习资源，帮助开发者理解 AI Agent 的核心架构
- **实验** — 探索 AI Agent 的创意应用，推动技术边界的边界

## 🚀 我们的项目

### [AIbubu](https://aibubu.app)

**AI 时代的编码计步器**

桌面宠物应用，监控你使用 AI 编码工具（Cursor、Claude Code、Codex、Trae）的活跃度，将其量化为步数，驱动桌面宠物行走 — 编码越多，宠物跑得越快。

[→ 了解更多](https://aibubu.app) · [GitHub](https://github.com/funAgent/ai-bubu)

---

### [Build Claude Code](https://build.funagent.app)

**从零构建企业级 AI Agent CLI**

49 节渐进式课程，教你从 `npm init` 开始，逐课构建出一个架构对标 Claude Code 的完整 AI Agent CLI 产品。

[→ 开始学习](https://build.funagent.app) · [GitHub](https://github.com/funAgent/build-claude-code-cli)
